
# YouTube Video Downloader MP3


## Installation

Install the required libraries:

     pip install pytube
   
## Usage

1. Run the program.

2. Enter the YouTube video URL you want to download.

3. The video will be downloaded and saved as an MP3 audio file.

## Requirements

- Python 
- pytube library
